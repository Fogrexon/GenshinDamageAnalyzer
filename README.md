# Genshin Damage Analyzer

Author: Fogrex

## About

原神の画面を解析して出たダメージを解析するアプリケーション。ダメージ以外にもフィールドボス認識機能もあり。

## Technologies

Damage Detection: OpenCV

Enemy Detection: YOLOX

Frontend: Vite + React
